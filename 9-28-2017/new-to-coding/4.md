##2. More exercises

Create a map of languages and play around with get
and put.

A map maps keys to values.

```
Key     Value
"CS"     "C#"
"GO"     "golang"
"JS"     "javascript"
"PY"     "python" 
```

```
package main

import (
	"fmt"
)

func main() {
	fmt.Println("Maps")
	
	var lang = make(map[string]string)
	lang["cs"] = "C #"
	lang["js"] = "javascript"
	lang["rb"] = "ruby"
	lang["go"] = "golang"
	
	fmt.Println("Value of key go = ",lang["go"])	
}
```

Can we now change the value of key "cs" to "csharp" 
and print?
